### Project Description
This project focuses on capturing and analyzing network traffic using Wireshark. Different protocols such as DNS, TCP, TLS, IPv4 and IPv6 were analyzed to understand how devices communicate over a network.

### Tools Used
- Wireshark
- tcpdump
- Snort

### Outcome
Successfully captured and analyzed network packets and studied various networking protocols.
